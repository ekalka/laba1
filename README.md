Блоки:

human-body (основной блок)
head (голова)
torso (туловище)
arms (руки)
legs (ноги)
Элементы:

head__eyes (глаза)
torso__heart (сердце)
arms__hand (кисть)
legs__foot (ступня)
Модификаторы:

head__tattoo_left (татуировка на левой стороне головы)
arms__ring_right (кольцо на правой руке)
legs__shoes_sport (спортивная обувь на ногах)
